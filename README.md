# Finance AI Agent using OpenAI Agents SDK and Ollama

## Overview

This project demonstrates how to build a simple AI-powered finance assistant using the **OpenAI Agents SDK**, **Ollama**, and **Yahoo Finance**.

Instead of hardcoding business logic inside prompts, the agent is equipped with Python tools that retrieve live financial data whenever required. When a user asks a question, the language model decides which tool to call, executes it, and generates a natural language response.

The project is intentionally small so that it serves as a good starting point for learning how tool-calling agents work.

---

## Features

- Runs completely with a local LLM through Ollama
- Uses the OpenAI Agents SDK for agent orchestration
- Retrieves live stock prices using Yahoo Finance
- Fetches analyst recommendations
- Demonstrates function/tool calling
- Easy to extend with additional financial tools

## How It Works

### 1. Configure Ollama

The notebook configures the OpenAI Agents SDK to communicate with a locally running Ollama server instead of OpenAI's cloud API.

```python
OPENAI_BASE_URL=http://localhost:11434/v1
OPENAI_API_KEY=ollama
```

---

### 2. Define Tools

The project defines two Python functions and exposes them as tools using the `@function_tool` decorator.

#### Current Stock Price

Retrieves the latest closing price for a stock.

Example:

```
What is the stock price of TSLA?
```

#### Analyst Recommendations

Retrieves the latest analyst recommendations available on Yahoo Finance.

Example:

```
Show analyst recommendations for AAPL.
```

---

### 3. Create the Agent

The Finance Agent is configured with:

- a name
- instructions
- an LLM
- a list of available tools

The language model automatically decides when one of these tools should be used.

---

### 4. Execute the Agent

A user prompt is passed to the agent.

```python
result = await Runner.run(
    finance_agent,
    "What is the current stock price for TSLA?"
)
```

If the model determines that external information is needed, it invokes the appropriate tool before generating the final answer.

## Why Use the OpenAI Agents SDK?

Instead of manually deciding which function to call, the SDK allows the language model to choose the appropriate tool based on the user's request.

This keeps the application logic simple while making the agent more flexible and easier to extend.
